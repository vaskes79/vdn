# Управление заметками (редактирование и удаление)

## Описание
Пользователь может редактировать текст существующих заметок или удалять их.

## Редактирование заметки

### Шаг 1: Открытие диалога редактирования
- Пользователь нажимает на иконку редактирования (карандаш) рядом с заметкой в списке
- Открывается модальное окно "Change Description Note"
- Воспроизведение видео останавливается

### Шаг 2: Изменение текста
- В диалоге отображается текущий текст заметки и временная метка
- Пользователь редактирует текст в поле ввода

### Шаг 3: Сохранение изменений
- Пользователь нажимает кнопку "OK"
- Происходит:
  1. Обновляется запись заметки в базе данных (таблица `NOTES`)
  2. Список заметок обновляется
  3. Диалог закрывается
  4. Воспроизведение видео возобновляется

### Отмена редактирования
- Пользователь может нажать кнопку "Cancel"
- Изменения отменяются, диалог закрывается, воспроизведение возобновляется

## Удаление заметки

### Шаг 1: Открытие диалога подтверждения
- Пользователь нажимает на иконку удаления (корзина) рядом с заметкой
- Открывается диалог подтверждения удаления
- Воспроизведение видео останавливается

### Шаг 2: Подтверждение удаления
- В диалоге отображается информация о заметке:
  - Временная метка (заголовок)
  - Текст заметки (описание)
- Пользователь подтверждает удаление нажатием кнопки "OK"
- Происходит:
  1. Заметка удаляется из базы данных
  2. Список заметок обновляется
  3. Диалог закрывается
  4. Воспроизведение видео возобновляется

### Отмена удаления
- Пользователь может нажать кнопку "Cancel"
- Удаление отменяется, диалог закрывается, воспроизведение возобновляется

## Технические детали

### Компоненты
- `Notes/NoteItem.js` - элемент списка заметок с кнопками действий
- `Dialog/Edit.js` - диалог редактирования (используется для заметок и видео)
- `Dialog/Confirm.js` - диалог подтверждения удаления

### Методы БД
- `db.editNote(id, title)` - обновление текста заметки
- `db.removeNote(id)` - удаление заметки из таблицы `NOTES`

### Состояние
- После редактирования или удаления вызывается `update()` для обновления списка заметок
- Все изменения сохраняются только в локальной базе данных (IndexedDB)
