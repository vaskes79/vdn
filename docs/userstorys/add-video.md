# Добавление нового видео

## Описание

Пользователь может добавить новое видео в приложение для последующего просмотра и создания заметок.

## Сценарий использования

### Шаг 1: Открытие диалога добавления видео

- Пользователь нажимает на кнопку с иконкой "+" (FAB) в правом верхнем углу навигационной панели
- Открывается модальное окно "Add Video"
- Воспроизведение текущего видео останавливается

### Шаг 2: Заполнение формы

- Пользователь вводит название видео в поле "Title Video"
- Пользователь вводит URL видео в поле "Url Video"
- Поддерживаются URL форматов:
  - YouTube: `https://youtu.be/...` или `https://www.youtube.com/watch?v=...`
  - Vimeo: `https://vimeo.com/...`

### Шаг 3: Подтверждение добавления

- Пользователь нажимает кнопку "Add Video"
- Происходит:
  1. Видео добавляется в базу данных (таблица `LIST_VIDEO`)
  2. Устанавливается как текущее активное видео (обновляется настройка `current_video`)
  3. URL видео устанавливается в проигрыватель
  4. Обновляется список видео в боковой панели
  5. Диалог закрывается
  6. Воспроизведение нового видео начинается автоматически

### Отмена добавления

- Пользователь может нажать кнопку "Cancel" или закрыть диалог
- В этом случае форма очищается, диалог закрывается, воспроизведение текущего видео возобновляется

## Технические детали

### Компоненты

- `Dialog/AddVideo.js` - диалог добавления видео
- Использует контекст `VdnAppContext` для доступа к методам БД

### Методы БД

- `db.addVideo(video)` - добавление видео в таблицу `LIST_VIDEO`
- `db.setCurrentVideo(url)` - установка текущего видео в настройках

### Состояние

- После добавления вызывается `update()` для обновления данных во всех компонентах
- Обновляется `urlVideo` в контексте приложения
