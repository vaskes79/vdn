# Экспорт заметок в разных форматах

## Описание
Пользователь может экспортировать список заметок для текущего видео в различных форматах: HTML, Markdown или обычный текст. Экспортированные данные копируются в буфер обмена.

## Сценарий использования

### Шаг 1: Выбор формата экспорта
- Пользователь просматривает список заметок для текущего видео
- В верхней части панели заметок находятся три кнопки:
  - **HTML** - экспорт в формате HTML
  - **TEXT** - экспорт в текстовом формате
  - **MARKDOWN** - экспорт в формате Markdown

### Шаг 2: Экспорт в выбранном формате
- Пользователь нажимает на кнопку с нужным форматом
- Происходит:
  1. Все заметки текущего видео форматируются в выбранном формате
  2. Форматированный текст копируется в буфер обмена
  3. Пользователь получает уведомление (через стандартное поведение браузера)

### Форматы экспорта

#### HTML формат
- Заметки форматируются как HTML список (`<ul>`)
- Каждая заметка представляет собой элемент списка (`<li>`) со ссылкой
- Ссылка содержит временную метку и текст заметки
- URL ссылки включает временную метку для перехода к моменту видео

Пример:
```html
<ul>
    <li><a href="https://www.youtube.com/watch?t=123&v=...">00:02:03 Текст заметки</a></li>
    <li><a href="https://www.youtube.com/watch?t=456&v=...">00:07:36 Другая заметка</a></li>
</ul>
```

#### Markdown формат
- Заметки форматируются как маркированный список Markdown
- Каждая заметка содержит временную метку в обратных кавычках и текст
- В конце добавляются ссылки-определения для каждой временной метки

Пример:
```markdown
- [`00:02:03`] Текст заметки
- [`00:07:36`] Другая заметка
[`00:02:03`]: https://www.youtube.com/watch?t=123&v=...
[`00:07:36`]: https://www.youtube.com/watch?t=456&v=...
```

#### Text формат
- Заметки форматируются как простой текст
- Каждая строка содержит временную метку, текст заметки и URL с временной меткой

Пример:
```
00:02:03 Текст заметки https://www.youtube.com/watch?t=123&v=...
00:07:36 Другая заметка https://www.youtube.com/watch?t=456&v=...
```

### Шаг 3: Использование экспортированных данных
- Пользователь может вставить скопированные данные в любой текстовый редактор, документ или другое приложение
- В HTML и Markdown форматах ссылки позволяют перейти к соответствующему моменту видео

## Технические детали

### Компоненты
- `Notes/Controls.js` - панель с кнопками экспорта
- Использует скрытое текстовое поле для копирования в буфер обмена

### Утилиты (`src/components/utils.js`)
- `formatToHtml(notes)` - форматирование в HTML
- `formatToMarkdown(notes)` - форматирование в Markdown
- `formatToText(notes)` - форматирование в текст
- `formatTime(duration)` - форматирование времени в `HH:MM:SS`
- `getVideoUrl(url, time)` - генерация URL с временной меткой
- `copyToClipBoard(el)` - копирование в буфер обмена через `document.execCommand('copy')`

### Особенности
- Экспортируются только заметки для текущего активного видео
- Временные метки форматируются в читаемый формат `HH:MM:SS`
- URL генерируются с учетом типа платформы (YouTube или Vimeo)
