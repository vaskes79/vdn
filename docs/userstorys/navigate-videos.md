# Навигация между видео

## Описание
Пользователь может переключаться между добавленными видео, просматривать список всех видео и управлять ими через боковую панель.

## Открытие боковой панели

### Шаг 1: Открытие списка видео
- Пользователь нажимает на иконку меню (гамбургер) в левом верхнем углу навигационной панели
- Открывается боковая панель (Drawer) со списком всех добавленных видео
- Воспроизведение текущего видео останавливается

### Шаг 2: Просмотр списка видео
- В боковой панели отображается список всех видео
- Каждый элемент списка содержит:
  - Название видео
  - Кнопки редактирования и удаления

## Переключение между видео

### Выбор видео из списка
- Пользователь нажимает на элемент списка с нужным видео
- Происходит:
  1. Выбранное видео устанавливается как текущее (обновляется настройка `current_video`)
  2. URL видео устанавливается в проигрыватель
  3. Загружаются заметки для выбранного видео
  4. Боковая панель автоматически закрывается
  5. Видео готово к воспроизведению

## Управление видео из боковой панели

### Редактирование названия видео
- Пользователь нажимает на иконку редактирования (карандаш) рядом с видео
- Открывается диалог "Change Title"
- Пользователь изменяет название и подтверждает изменения
- Название обновляется в базе данных и в списке

### Удаление видео
- Пользователь нажимает на иконку удаления (корзина) рядом с видео
- Открывается диалог подтверждения удаления
- При подтверждении:
  1. Видео удаляется из базы данных
  2. Все связанные заметки также удаляются
  3. Устанавливается демо-видео по умолчанию
  4. Список видео обновляется
  5. Боковая панель остается открытой

## Закрытие боковой панели

- Пользователь может закрыть панель:
  - Нажав на иконку меню еще раз
  - Кликнув вне области панели
  - Нажав клавишу Escape
- При закрытии воспроизведение видео возобновляется

## Переход к моменту видео через заметку

### Использование заметок для навигации
- Пользователь просматривает список заметок в правой части интерфейса
- Каждая заметка отображает:
  - Временную метку (в формате `HH:MM:SS`)
  - Текст заметки
- Пользователь кликает на заметку
- Проигрыватель автоматически переходит к моменту видео, соответствующему временной метке заметки
- Видео начинает воспроизведение с этого момента

## Технические детали

### Компоненты
- `Sidebar/Sidebar.js` - боковая панель со списком видео
- `Sidebar/SidebarList.js` - список элементов видео
- `Sidebar/SidebarItem.js` - элемент списка с действиями
- `Notes/NoteItem.js` - элемент заметки с возможностью перехода

### Методы БД
- `db.getVideoList()` - получение списка всех видео
- `db.setCurrentVideo(url)` - установка текущего видео
- `db.getNoteList(url)` - получение заметок для конкретного видео
- `db.editVideo(urlID, title)` - редактирование названия видео
- `db.removeVideo(urlID)` - удаление видео и связанных заметок

### Состояние
- При переключении видео обновляется `urlVideo` в контексте
- Вызывается `update()` для обновления данных во всех компонентах
- Заметки автоматически фильтруются по URL текущего видео

### Особенности
- При удалении видео автоматически удаляются все связанные заметки через курсор IndexedDB
- После удаления видео устанавливается демо-видео по умолчанию
- Боковая панель использует Material-UI Drawer компонент
- При открытии/закрытии панели управляется состояние воспроизведения видео
