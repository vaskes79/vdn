# Навигация по видео через заметки

## Описание

Пользователь может перемещаться по видео, кликая на заметки в списке. При клике на заметку видеоплеер автоматически перематывается к соответствующему таймкоду, что позволяет быстро просматривать помеченные фрагменты видео.

## Сценарий использования

### Шаг 1: Просмотр списка заметок

- Пользователь видит список заметок в правой панели приложения
- Каждая заметка отображает:
  - Временную метку (таймкод) в формате MM:SS
  - Текст заметки (описание)

### Шаг 2: Переход к таймкоду заметки

- Пользователь кликает на любую заметку в списке (на область с временной меткой и текстом)
- Происходит:
  1. Видеоплеер автоматически перематывается к таймкоду, указанному в заметке
  2. Воспроизведение видео продолжается с новой позиции (если видео было запущено)
  3. Пользователь видит фрагмент видео, соответствующий выбранной заметке

### Шаг 3: Навигация между заметками

- Пользователь может последовательно кликать на разные заметки
- Каждый клик перематывает видео к соответствующему таймкоду
- Это позволяет быстро перемещаться между помеченными фрагментами видео

## Технические детали

### Компоненты

- `Notes/NoteItem.tsx` - элемент списка заметок с обработчиком клика
- `Video/PlayerContext.tsx` - контекст для доступа к методам видеоплеера
- `Video/VideoPlayer.tsx` - компонент видеоплеера с методом `seekTo`

### Методы

- `usePlayer().seekTo(seconds)` - перемотка видеоплеера к указанному времени в секундах
- `note.time` - таймкод заметки в секундах (из типа `Note`)

### Состояние

- Перемотка происходит мгновенно без остановки воспроизведения
- Если видео было на паузе, оно остается на паузе после перемотки
- Если видео воспроизводилось, оно продолжает воспроизведение с новой позиции
